# Learning Resources

## Purpose

The **Learning Resources** library is a curated collection of books, courses, videos, laboratories, tools, standards, research papers, and community resources that support the OT/ICS Cybersecurity Roadmap.

Unlike traditional "awesome" repositories, this library does not attempt to collect every available resource.

Instead, it focuses on **quality over quantity**, recommending resources that provide the greatest educational value while aligning with the curriculum.

The objective is to answer four questions for every learner:

* What should I learn?
* Why should I learn it?
* When should I learn it?
* Where should I learn it?

---

# Educational Philosophy

Every resource included in this library has been selected because it contributes to one or more learning objectives within the curriculum.

Resources are intended to complement—not replace—the curriculum.

Learners are encouraged to:

1. Learn the concepts.
2. Study the recommended resources.
3. Complete the associated labs.
4. Build the recommended projects.
5. Validate their understanding through self-assessments.

# Resource Tiers

Resources are classified into three tiers.

## Core

Core resources are considered essential.

Every learner following the curriculum is expected to complete these resources.

Examples include:

* NIST SP 800-82
* MITRE ATT&CK for ICS
* Mike Holcomb
* Cisco Networking Academy
* Wireshark Documentation

---

## Recommended

Recommended resources provide additional depth and practical experience.

These resources are strongly encouraged but are not mandatory.

Examples include:

* RealPars
* PLC Dojo
* Practical Networking
* David Bombal
* Inductive Automation University

---

## Reference

Reference resources are intended for deeper study, research, or vendor-specific implementation details.

Examples include:

* Vendor documentation
* Academic papers
* RFCs
* Product manuals
* Protocol specifications
