# OT/ICS Cybersecurity Community

## Purpose

OT/ICS cybersecurity is a multidisciplinary field built by engineers, researchers, educators, standards organizations, vendors, and open-source contributors.

This document serves as a guide to the broader OT/ICS cybersecurity ecosystem, helping learners discover the people, organizations, events, publications, and communities that shape the industry.

This is not a ranking. It is a curated reference intended to encourage exploration and continuous learning.

# Landmark Incidents

Every OT security practitioner should study these incidents.

* Stuxnet
* BlackEnergy
* Industroyer / CrashOverride
* TRITON / TRISIS
* Colonial Pipeline
* Oldsmar Water Treatment Facility
* PIPEDREAM / INCONTROLLER
* FrostyGoop

Each incident should be understood from both a technical and operational perspective, including initial access, attack progression, impact, detection opportunities, and defensive lessons.
