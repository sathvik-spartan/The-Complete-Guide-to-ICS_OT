# Learning Paths

## Basic Overview

The OT/ICS Cybersecurity Roadmap is designed for learners from diverse technical backgrounds.

While the curriculum remains the same, not every learner needs to study every module in the same order. This document provides recommended learning paths based on your existing experience.

If you are unsure which path to follow, begin with the **Beginner Path**.

# Learning Recommendations

To get the most out of this curriculum:

1. Complete the modules in the recommended order.
2. Prioritize __official documentation__ over secondary sources.
3. Complete every lab before moving to the next module.
4. Build every project and publish your work along with professional documentation of the project
5. Maintain notes throughout your learning journey.
6. Revisit earlier modules as your understanding improves.

---

# Choosing a Path

If you are uncertain which learning path best matches your background, i would personally suggest beginning with the **Beginner Path** as it provides the strongest foundation and ensures that later topics are build upon well-understood concepts.
